# End-To-End-Project-using-Scikit-Learn-ML-Library

## The project consists of the following stages:
- Preprocessing
    - Data visualization
    - Feature selection
    - Data normalization
    - Data training-test stratified split
- Model training
- Model evaluation
- Model fine-tuning
- Model evaluation post fine-tuning

## ML models used:
- SGD Regressor
- Random Forest Regressor
- MLP Regressor
